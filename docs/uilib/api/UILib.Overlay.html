SetAutoPause(bool)

Set whether this overlay should automatically pause the game when it's shown.

Calling with autoPause being false:

  • If the overlay has an active PauseHandle, it will be closed immediately regardless of its visibility.

Calling with autoPause being true:

  • If the overlay is visible, a new PauseHandle will be allocated.
  • If the overlay is hidden, no PauseHandle will be allocated.
public void SetAutoPause(bool autoPause)

Parameters

autoPause bool

Whether to pause the game automatically

SetSortable(bool)

Toggles the ability for this overlay to move to the front above all other overlays.
